# 🧠 NeuroPlan AI – Adaptive Study Intelligence System

## 🚀 Overview

**NeuroPlan AI** is an advanced, AI-inspired web application designed to solve one of the most common real-world problems faced by students: **ineffective study planning and time management**.

Unlike traditional planners, NeuroPlan AI acts as a **smart study coach** that dynamically adapts to user input, analyzes behavior, and generates personalized study strategies.

---

## 🎯 Problem Statement

Students often struggle with:

* Poor time management
* Lack of structured study plans
* Procrastination
* Uneven focus across subjects
* Burnout due to inefficient scheduling

This project addresses these issues by creating an **intelligent, adaptive planning system**.

---

## 💡 Key Features

### 🧾 1. Dynamic Subject Input

* Add multiple subjects
* Define:

  * Difficulty level (Easy / Medium / Hard)
  * Days left for exam
  * Preparation percentage

---

### 🧠 2. Smart Study Plan Generator

* Automatically generates a **personalized daily study plan**
* Assigns:

  * Priority (HIGH / MEDIUM / LOW)
  * Recommended study hours

---

### 🔍 3. Weakness Detection System

* Identifies the **weakest subject**
* Highlights areas needing improvement

---

### 📊 4. Productivity Score

* Calculates performance based on preparation levels
* Displays a **real-time productivity score**

---

### 📅 5. Daily Dynamic Planner

* Generates a **clean, structured “Today’s Plan”**
* Updates dynamically based on inputs

---

### ⏱️ 6. Focus Mode (Pomodoro Timer)

* 25-minute study sessions
* Start / Pause / Reset functionality
* Encourages deep work and focus

---

### 🎮 7. Gamification System

* Study streak tracking
* Points-based motivation system
* Enhances engagement and consistency

---

### ⚠️ 8. Burnout Detection

* Alerts user if:

  * Overworking (> 8 hours)
  * Underperforming (< 2 hours)

---

### 📈 9. Analytics Dashboard

* Visual insights using charts:

  * Subject priority distribution
  * Study time allocation
* Helps in understanding study patterns

---

### 🔮 10. Prediction Engine

* Estimates:

  > “Syllabus completion time based on current pace”

## ▶️ How to Run the Project

### 🔹 Option 1: Without Backend (Frontend Only)

1. Download or clone the repository
2. Open `index.html` in your browser

---

### 🔹 Option 2: With Flask Backend

1. Install dependencies:

```
pip install flask
```

2. Run the server:

```
python app.py
```

3. Open in browser:

```
http://127.0.0.1:5000
```
